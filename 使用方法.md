# 🕵️‍♂️ FakeSimSun 字体混淆工具

本工具可将正常文本转换为“视觉相同但编码不同”的混淆文本。  
配合自定义字体 **`FakeSimSun`**，实现 **“看起来正常，复制即乱码”** 的效果，适用于防复制、趣味排版等场景。

---

## 📦 使用步骤

### 1\. 安装混淆字体

* 双击 `fakesimsun.ttf`（或 `fakesimsun.ttc`）文件；
* 点击 **“安装”**（Windows）或 **“安装字体”**（macOS）；
* 安装后，系统将识别名为 **`FakeSimSun`** 的新字体。

> 💡 提示：若文件名为 `.ttc`，Windows 仍可正常安装。

---

### 2\. 打开转换工具

* 在浏览器中打开 **`index.html`**（建议使用 Chrome、Edge 或 Firefox）；
* 工具页面包含两个功能区：

  * **正向转换**：输入正常文本 → 生成混淆文本；
  * **反向还原**：粘贴混淆文本 → 还原原始内容。

---

### 3\. 复制并使用混淆文本

* 点击 **“转为混淆文本”**，然后点击 **“复制结果”**；
* 将文本粘贴到 **Word、PPT、网页编辑器等任意位置**；
* 此时文本显示为 **乱码或异常字符** —— ✅ 这是正常现象！

---

### 4\. 应用字体查看真实内容

* 在目标软件（如 Microsoft Word）中：

  1. 选中粘贴的混淆文本；
  2. 将字体设置为 **`FakeSimSun`**；
  3. 文本将立即显示为你最初输入的 **正常内容**！

> 🔒 效果说明：他人若未安装 `FakeSimSun` 字体，复制后只能看到乱码，无法获取原文。

---

## 🛠️ 技术说明

* 字体映射关系由 `char\\\\\\\_mapping.txt` 定义；
* `index.html` 内嵌了完整的编码映射（由 `gen\\\\\\\_js\\\\\\\_map.py` 生成）；
* 所有处理均在浏览器本地完成，**无需联网，保护隐私**。

---

## ⚠️ 注意事项

* **字体版权**：`fakesimsun.ttf` 基于 SimSun 修改，请勿用于商业分发。
* **跨平台兼容性**：

  * Windows 用户体验最佳；
  * macOS / Linux 用户需手动安装字体才能正确预览。

* 若在 Word 中仍显示异常，请确认：

  * 字体名称是否为 **`FakeSimSun`**（非 `SimSun`）；
  * 是否已重启 Word（部分版本需重启才能识别新字体）。

## 🧩 项目文件说明

| 文件 | 说明 |
|------|------|
| `fakesimsun.ttf` | 混淆字体文件（需安装） |
| `simsun.ttc` | 正常宋体字体（用于避免误操作导致原字体出错） |
| `index.html` | 核心转换工具（双击即可使用） |
| `mapping.js` | （开发者用）工具所使用的字符映射表 |
| `char\\\\\\\_mapping.txt` | （开发者用）字符映射表（供调试或重新生成 JS） |
| `convert.py` | （开发者用）将映射表转为 JavaScript |
| `fakefont.py` | （开发者用）制作假字体 |
| `name.py` | （开发者用）命名字体 |
